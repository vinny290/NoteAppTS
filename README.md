1. 
   - **Клонировать репозиторий:**
     ```bash
     git clone https://github.com/vinny290/noteAppTs.git
     ```
2. 
   - **Сервер:**
     ```bash
     cd server
     npm install
     npm run server
     cd ..
     ```
3.
   - **Клиент:**
     ```bash
     cd client
     npm install
     npm run dev
     ```
